Web sayt haqida

Web-sayt o'qitayotgan talabalarimga Web-sahifa yaratishni osonlashtirish maqsadida yaratilgan bo'lib, sayt tuzilishini Visual studio code orqali o'rganish uchun xizmat qiladi.Web-sayt har dars boshlanishidan oldin yengilanib turadi. Web-sayt fayllarini ko'zdan kechirib, qayerda nima joylashganligini o'rganing

Web saytagi narsalarning joylashuvi

Asosiy nav blokida har bir o'o'rganganlarimiz to'g'risida qisqacha ma'lumot berilgan va undan tashqari yon tomondagi (o'ng tarafda) joylashgan knopka orqali dars davomida o'tilgan mavzular haqida ma'lumot olishingiz mumkin

Web sahifamizani qonun qoidalari

  1. Web-saytdagi (teglar va css xususiyatlari) qisqa ma'lumotlar daftariga yozilgan bo'lishi shart
  2. Web-saytga kirishdan oldin Git orqali web-sayt yangilangan yoki yengilanmaganligini tekshirib ko'ring (git pull komandasi orqali)
  3. Web-saytdagi materiallarni tarqatish(ko'chirma olish) yoki ulardan foydalangan holda boshqalarni o'qitish qatiyan man etiladi
  4. Git orqali saytga o'zgartirishlar kiritish mumkin emas!
