P elementiga <p style="color:red;"> berilgan, headni ichida esa <style>p{color:blue;}</style> qo'yilgan va undan tashqari, tashqi fayl qilib p elementiga link orqali <link rel="stylesheet" href="style.css" p{color:green;} berilgan. Lekin eng yaqin element ta'sir ko'rsatib bu yozuv qizil ko'rinmoqda

Bu yerda tepadigidek blok qilingan. Faqatgina farqi p ichida style atributi o'chirilgan, rangi ko'kka almashtirilgan( <style> tegini ichidigi ishlayapti).style tegini ichiga yozilgan narsalar birinchi keladi deb o'ylashingiz mumkin,lekin unday emas! link ichida va style tegi ichida yozilgan narsalar bir xil darajada turadi, faqatgina kod ketma-ketligi har xil, link birinchi yozilgan, style esa keyin. Ohirgi yozilgan qonun ko'proq ta'sir o'tkazadi!
